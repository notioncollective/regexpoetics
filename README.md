regexpoetics
============

A simple interface for turning regular expression patterns into osc messages.

## Instructions

Install npm dependencies:

	npm install

Install bower dependencies:

	bower install

Start the server:

	grunt

Open up your browser to [http://localhost:3000](http://localhost:3000)
